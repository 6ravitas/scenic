# Scenic.Math

[![Build Status](https://travis-ci.org/boydm/scenic_math.svg?branch=master)](https://travis-ci.org/boydm/scenic_math)

Math support for Scenic applications

## Installation

Add `scenic_math` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:scenic_math, "~> 0.7.0"}]
end
```
